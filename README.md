# AI Task Manager

Ye ek simple task manager project hai jo maine internship assignment ke liye banaya hai.
Frontend React me hai aur backend Node.js + Express me.

## Features

- Task add kar sakte hain
- Task complete mark kar sakte hain
- Task delete kar sakte hain
- Upar se kuch simple suggestions bhi show ho jati hain (static)

## How to run

### Backend
cd backend  
npm install  
node index.js  

### Frontend
cd frontend  
npm install  
npm start
